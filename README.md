# Simple React App

##### How to use it

1. Clone or zip file from github repo.
2. Open with prefered text editor. 
3. Run cmd

```
npm install
```
4. Then cmd
```
npm start
```
5. Open localhost:3000 and view welcome page. You are all set to go.
